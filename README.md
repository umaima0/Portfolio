## 🛰️ Overview

Welcome to my personal web portfolio. This is a highly interactive, responsive, and tactile single-page application built using clean, semantic HTML5, modern CSS Grid/Flexbox layouts, and optimized vanilla JavaScript. It is styled with a custom-engineered dark maroon aesthetic, complete with a dynamic HUD progress tracker, scroll-activated reveals, and custom 3D card tilt effects.

## 📂 Featured Projects

### 01. Space Explorer
*   **Description:** A collaborative web application designed to track real-time rocket launches and maintain custom celestial logs. Pulls dynamic data payloads from NASA's APOD and SpaceDevs APIs, integrated with Stellarium Web and Geolocation APIs to render what is currently overhead based on local coordinates.
*   **Stack:** MongoDB, Mongoose, Node.js, Express, SpaceDevs API, NASA APOD API.
*   **Codebase:** [Space Explorer 2.0](https://github.com/fazasfz/SPACEEXPLORER/tree/main/SpaceExplorer2.0)

### 02. Smart Recipe Finder
*   **Description:** An AI-driven search and recommendation engine. Utilizes BFS and A* search algorithms to parse recipe paths under ingredient constraints, alongside K-Nearest Neighbors (KNN) and K-Means clustering algorithms to recommend alternative matches from a real Kaggle food dataset.
*   **Stack:** Python, Flask, Streamlit, Scikit-learn, Pandas, NumPy.
*   **Codebase:** [Recipe Engine](https://github.com/umaima0/Recipe-Engine)

### 03. HappiTail — Virtual Pet Simulator
*   **Description:** A real-time, desktop virtual pet simulator inspired by classic Tamagotchi behaviors. Designed with species-specific actions, discrete pet stats (hunger, mood, energy), pixel-art rendering pipelines, and an interactive interface. Structured entirely with Object-Oriented Programming (OOP) in C++ to achieve modular state-based management.
*   **Stack:** C++, SFML Library.
*   **Codebase:** [HappiTail](https://github.com/umaima0/HappiTail-Virtual-Pet-Stimulator)
