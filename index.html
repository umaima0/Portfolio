PROJECT 01

Space Explorer

A web app for tracking live rocket launches and logging personal sky observations. I built the Live Launch Tracker and Observation Log modules — pulling real-time data from the SpaceDevs API and NASA's Astronomy Picture of the Day, and combining Stellarium Web with geolocation to surface what's actually visible in the sky from wherever you are. The backend runs on MongoDB, with schema design and aggregation pipelines handling the heavier querying.

  • MongoDB
  • Mongoose
  • SpaceDevs API
  • NASA APOD API
  • Stellarium Web
  • Geolocation API
